Project Summary

In this project, weather data from the OpenWeather API was used to study weather patterns across the world's latitudes. This data was then used to answer the question, "How does weather change as we approach the equator?".

The parameters researched were as follows: Maximum Temperature, Humidity, Cloudiness, and Wind Speed. You can use the navigation bar or the figures to the right to examine each parameter separately, as well as review the full analysis.
